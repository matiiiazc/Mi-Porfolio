hablidades

Manejo bastante bien HTML; sé crear formularios con inputs, anchors, clases, secciones, etiquetas semánticas, y puedo estructurar un código claro, ordenado y fácil de mantener.

Con CSS aplico estilos modernos y organizados. Sé manejar propiedades de color, tipografía, márgenes, paddings, y crear diseños responsivos con flexbox y grid. También puedo animar elementos y lograr una buena estética visual.

En JavaScript puedo agregar interactividad a mis proyectos, validar formularios, manipular el DOM, crear funciones, trabajar con eventos y conectar el frontend con el backend usando fetch y JSON.

Con Python desarrollo scripts, programas y lógicas de backend. Sé trabajar con funciones, bucles, condicionales, listas, diccionarios y archivos. También he practicado conceptos de programación orientada a objetos.

En SQL sé crear bases de datos, tablas y relaciones. Puedo hacer consultas con SELECT, INSERT, UPDATE y DELETE, así como aplicar filtros, condiciones y uniones entre tablas.

Con MySQL administro bases de datos desde su interfaz y desde consola. Puedo crear usuarios, gestionar permisos, importar o exportar datos y conectar bases de datos con aplicaciones web.
